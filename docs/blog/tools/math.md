## Math

```js

//算开方
Math.sqrt(4)  //2   

//取整
Math.trunc(3.14)  //3

//函数返回一个数字的符号, 指示数字是正数，负数还是零
Math.sign() 

Math.sign(3);     //  1
Math.sign(-3);    // -1
Math.sign("-3");  // -1
Math.sign(0);     //  0
Math.sign(-0);    // -0
Math.sign(NaN);   // NaN
Math.sign("foo"); // NaN
Math.sign();      // NaN

//算次方
Math.pow(2, 3)  //8   2^3

```
